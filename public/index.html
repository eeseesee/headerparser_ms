Header Parser Microservice

Full stack Javascript app built with node.js, express.js, and Bootstrap by ECC

User stories:
    1) I can get the IP address, language and operating system for my browser.

Example usage:

https://ecc-headerparser-ms.herokuapp.com/api/whoami

Example output:

{ "ipaddress": "67.242.86.108", "language":"en-US", "software":"Macintosh; Intel Mac OS X 10_11_3"} }
